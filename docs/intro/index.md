# 开始阅读

`Cyber 2026` 是一套面向团队内部的赛博飞升方法论。它不试图承载真实业务功能，而是把团队实现“赛博飞升”所需的方法讲清楚：

- 为什么值得把更多工作交给 AI
- 什么任务该停在 prompt / rule / MCP / skill / hook
- 什么任务值得继续升级成 agent
- 如何把这些能力安全接入团队协作流程

## 这个站点怎么读

建议从 [范式转变](/paradigm-shift/) 开始，沿主线阅读到 [团队工作流与质量控制](/team-workflow/)。完整的内容分层、辅助资料入口和按目标选路径的建议，见 [内容地图](/project-overview/)。

## 本站不做什么

- 不承载业务专项 skill / agent 本体实现
- 不提供生产级 MCP 服务、项目私有配置、密钥和环境
- 不充当真实团队项目中的运行时资产仓
