## 如何使用本仓库

- **管理者：** 重点阅读第一章和第三章，理解团队引入 AI 工具的价值与路径
- **工程师：** 按顺序阅读，每章末尾有"完成检查清单"可自测掌握程度
- **模板文件：** `templates/` 目录下的配置文件可直接复制到自己的项目中使用
